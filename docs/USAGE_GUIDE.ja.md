## はじめに

TENJINは、200以上の教育理論をAIアシスタントが活用できるようにするMCPサーバーです。Claude、GPT、Geminiなどのアシスタントと連携して、教育理論に基づいた指導設計や研究支援を行います。

### 対象ユーザー

- **教員・教育者**: 授業設計、指導法の改善
- **研究者**: 教育理論研究、論文執筆
- **教育コンサルタント**: カリキュラム設計、研修開発
- **学生**: 教育学の学習、レポート作成

## 基本的な使い方

### 理論の取得

特定の理論の詳細を取得するには:

```
「構成主義について教えてください」
```

AIアシスタントは`get_theory`ツールを使用して、構成主義の詳細情報を取得します。

### 理論の一覧

カテゴリー別に理論を確認:

```
「学習理論のカテゴリーに含まれる理論を教えてください」
```

### 優先度での絞り込み

重要度の高い理論を取得:

```
「最も重要な教育理論（優先度5）を教えてください」
```

## 理論の検索

### キーワード検索

```
「アクティブラーニングに関連する理論を検索してください」
```

### 意味的検索（Semantic Search）

自然言語での検索:

```
「生徒の内発的動機づけを高めるための理論を探しています」
```

### ハイブリッド検索

グラフ構造とベクトル類似性を組み合わせた高度な検索:

```
「協同学習と関連性が高く、日本の教育現場で実践しやすい理論を探してください」
```

## 理論の分析

### 単一理論の分析

```
「自己決定理論について、中学校の数学授業での応用可能性を分析してください」
```

AIアシスタントは以下を含む詳細な分析を提供:
- 理論の概要
- 主要な原則
- 強み・限界
- 具体的な応用例

### 理論の比較

```
「構成主義と行動主義を比較してください」
```

比較結果には以下が含まれます:
- 共通点
- 相違点
- 統合的な視点
- 使い分けの推奨

### 複数理論の統合

```
「経験学習理論、省察的実践、足場かけ理論を統合して、効果的な実習指導の枠組みを提案してください」
```

## グラフ探索

### 関連理論の発見

```
「社会的構成主義に影響を与えた理論と、そこから派生した理論を教えてください」
```

### 理論間のパス探索

```
「行動主義から構成主義への理論的な発展の道筋を示してください」
```

### 理論ネットワークの可視化

```
「認知負荷理論を中心とした理論ネットワークを表示してください」
```

### 影響力の分析

```
「最も多くの理論に影響を与えている理論はどれですか？」
```

## 引用生成

### 単一理論の引用

```
「構成主義の引用をAPA形式で生成してください」
```

出力例:
```
Piaget, J. (1952). The origins of intelligence in children. 
International Universities Press.
```

### 参考文献リスト

```
「使用した理論の参考文献リストをAPA形式で作成してください」
```

### BibTeX/RISエクスポート

```
「選択した理論の引用をBibTeX形式でエクスポートしてください」
```

## 授業計画への応用

### 授業計画の作成

```
「高校2年生の英語のリーディング授業（50分）を、認知負荷理論とスキャフォールディングを活用して計画してください」
```

AIアシスタントは`lesson_plan`プロンプトを使用して、以下を含む授業計画を生成:

1. **学習目標**
2. **理論的根拠**
3. **導入（5分）**
4. **展開（35分）**
5. **まとめ（10分）**
6. **評価方法**
7. **教材・準備物**

### 学習者に適した理論の推薦

```
「学習意欲が低下している中学生への指導に役立つ理論を推薦してください」
```

### 学習パスの作成

```
「教育理論を体系的に学ぶための学習パスを作成してください。初心者向けで、3ヶ月程度で学べるものを希望します」
```

## 日本の教育文脈での活用

### 日本の教育への適応

```
「プロジェクト・ベースド・ラーニングを日本の小学校（総合的な学習の時間）で実践するための具体的なアプローチを提案してください」
```

AIアシスタントは`japanese_context`プロンプトを使用して、以下を考慮した提案を生成:

- 学習指導要領との整合性
- 日本の学校文化・慣行
- 評価観点との関連
- 具体的な実践例

### 授業研究との連携

```
「授業研究で「社会的構成主義」の視点から授業を分析するためのポイントを教えてください」
```

### 東洋の教育思想

```
「儒教教育思想と現代の教育理論の関連性について説明してください」
```

## ユースケース例

### 1. 論文執筆支援

```
「協同学習に関する研究論文を執筆しています。
関連する主要理論とその関係性をまとめ、
参考文献リストを作成してください」
```

### 2. 研修資料作成

```
「新任教員向けの研修で「形成的評価」について説明します。
理論的背景と具体的な実践方法をまとめた資料を作成してください」
```

### 3. カリキュラム設計

```
「中学校の理科カリキュラムを探究学習中心に再設計したいです。
活用できる理論と、学年ごとの発達段階を考慮した設計指針を提案してください」
```

### 4. 学習困難への対応

```
「読み書きに困難を抱える児童への指導に活用できる理論を教えてください。
特に、認知的な特性に配慮したアプローチを知りたいです」
```

### 5. テクノロジー統合

```
「タブレット端末を活用した数学の授業を設計したいです。
TPACKの観点から、効果的なテクノロジー統合の方法を提案してください」
```

## トラブルシューティング

### 検索結果が少ない場合

- より一般的なキーワードを使用
- 日本語/英語両方で検索
- カテゴリーフィルターを緩める

### 分析が浅い場合

- コンテキスト（対象学年、教科など）を明確に指定
- 「詳細に」「具体的に」などの指示を追加
- 複数の理論を組み合わせて分析を依頼

### 日本の文脈に合わない場合

- 「日本の教育現場で」と明示
- 学習指導要領との関連を質問
- 具体的な学校種・教科を指定
